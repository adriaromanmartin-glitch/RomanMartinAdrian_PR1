# Focus Garden - Instruccions d’ús

## Instal·lació

Executar les següents ordres:

npm install
npm run build
npx cap sync android
npx cap open android

## Funcionalitats principals

* Temporitzador de concentració
* Visualització generativa amb p5.js
* Mode nit
* Configuració desada localment

## Dispositiu de proves

* Terminal: Pixel 6
* Versió Android: 14.0
